## 四、系统优化建议

### 4.1 已优化项

1. **前端错误提示去重**：移除拦截器重复弹窗，提升用户体验
2. **通知机制完善**：借阅/预约/取书全流程通知覆盖
3. **排队位置维护**：所有状态变更场景均更新排队位置
4. **借阅资格校验**：取书借阅与正常借阅校验逻辑统一

### 4.2 后续优化建议

| 优先级 | 优化项                    | 说明                                |
| --- | ---------------------- | --------------------------------- |
| 高   | 添加乐观锁                  | Book 实体添加 @Version 字段，替代行级锁提升并发性能 |
| 高   | 添加 H2 测试依赖             | 支持不依赖 MySQL 的集成测试                 |
| 中   | 预约到书短信/邮件通知            | 当前仅站内通知，可扩展多渠道通知                  |
| 中   | 图书详情评论显示用户名            | 当前 ReviewItem 未填充 username 字段     |
| 中   | 前端预约列表分页               | 当前一次性加载全部预约，数据量大时需分页              |
| 低   | MetaObjectHandler 自动填充 | 配置 createdAt/updatedAt 自动填充处理器    |
| 低   | 操作日志 AOP 完善            | 记录借阅/预约等关键操作日志                    |

### 4.3 边界情况处理建议

1. **并发预约**：当前 reserve() 未加行级锁，高并发下排队位置可能重复，建议添加 `selectByIdForUpdate`
2. **多副本场景**：当前预约逻辑假设单副本，多副本时预约触发条件需调整为"所有副本均借出"
3. **预约过期清理**：当前过期任务每天凌晨 3 点执行，建议添加预约即将过期提醒

## 七、开发环境配置

### 7.1 JDK 版本兼容性

| 日期         | 问题                   | 描述                                                                                 | 解决方案                                                 |
| ---------- | -------------------- | ---------------------------------------------------------------------------------- | ---------------------------------------------------- |
| 2026-06-21 | Trae 沙箱默认 JDK 25 不兼容 | 沙箱 `JAVA_HOME` 固定为 `jdk-25.0.3`，项目依赖 Spring Boot 3.2.0 + Lombok 1.18.40，仅支持 JDK 17 | `start.bat` 启动脚本中强制设置 `JAVA_HOME` 和 `PATH` 指向 JDK 17 |

### 7.2 VS Code Java 语言服务器配置

为确保跨电脑移植无需手动修改 JDK 路径（前提：每台电脑已配置 `JAVA_HOME` 系统环境变量），在 `.vscode/settings.json` 中使用 `${env:JAVA_HOME}` 动态引用：

```json
{
    "java.compile.nullAnalysis.mode": "automatic",
    "java.configuration.updateBuildConfiguration": "automatic",
    "java.jdt.ls.java.home": "${env:JAVA_HOME}",
    "java.configuration.runtimes": [
        {
            "name": "JavaSE-17",
            "path": "${env:JAVA_HOME}",
            "default": true
        }
    ]
}
```

### 7.3 启动脚本 (`start.bat`)

在脚本开头强制指定 JDK 17，避免沙箱环境 JDK 优先级覆盖：

```bat
REM --- Force JDK 17 (Trae 沙箱默认 JDK 25，项目需要 JDK 17) ---
set JAVA_HOME=C:\Program Files\Java\jdk-17.0.5
set PATH=%JAVA_HOME%\bin;%PATH%
```
